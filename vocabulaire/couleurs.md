# Les couleurs

Une couleur se dit 색. Pour chaque couleur on le verra comme suffixe.

| Français | Coréen |
| -------- | ------ |
| Bleu     | 바란색 |
| Rouge    | 빨간색 |
| Jaune    | 노란색 |
| Vert     | 초록색 |
| Rose     | 분혹색 |
| Violet   | 보라색 |
| Orange   | 주황색 |
| Marron   | 갈색   |
| Blanc    | 하얀색 |
| Noir     | 검은색 |
| Gris     | 회색   |
